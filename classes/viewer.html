Class Viewer

class

Viewer

classdesc

The Viewer object represents the navigable photo viewer. Create a Viewer by specifying a container, client ID, photo key and other options. The viewer exposes methods and events for programmatic interaction.

setFilter

  • setFilter(filter: FilterExpression): when.Promise
    • Defined in viewer/Viewer.ts:386

    Set the filter selecting nodes to use when calculating the spatial edges.

    description

    The following filter types are supported:

    Comparison

    ["==", key, value] equality: node[key] = value

    ["!=", key, value] inequality: node[key] ≠ value

    ["<", key, value] less than: node[key] < value

    ["<=", key, value] less than or equal: node[key] ≤ value

    [">", key, value] greater than: node[key] > value

    [">=", key, value] greater than or equal: node[key] ≥ value

    Set membership

    ["in", key, v0, ..., vn] set inclusion: node[key] ∈ {v0, ..., vn}

    ["!in", key, v0, ..., vn] set exclusion: node[key] ∉ {v0, ..., vn}

    Combining

    ["all", f0, ..., fn] logical AND: f0 ∧ ... ∧ fn

    A key must be a string that identifies a node property name. A value must be a string, number, or boolean. Strictly-typed comparisons are used. The values f0, ..., fn of the combining filter must be filter expressions.

    Clear the filter by setting it to null or empty array.

    example

    viewer.setFilter(["==", "sequenceKey", "<my sequence key>"]);

    Parameters

    • filter: FilterExpression

      The filter expression.

    Returns when.Promise

    Promise that resolves after filter is applied.
